Welcome to BandWagonhost Page.

本人使用搬瓦工作为自己日常翻墙、练手后台的VPS，在网上看到有人用web 和android 写了客户端。由于原理比较简单，就用swift写了一个iOS客户端，并上传到appstore。方便开发者或者爱好者。

Usage 使用事项

  • 登录 https://bandwagonhost.com/ 并进入 My service -》 kiwivm control 面板。
  • KiwiVM Extras -> API ，在此页可以看到 veid ，并点击 show api key ,找到 api key。
  • 打开 《搬瓦工》app ，输入veid & apikey 登录。
    登录成功后，会记录你的veid和apikey ，veid 存储到 NSUserdefault 中，非安全。apikye 存储到 keychain中，如果手机支持指纹识别， 那么每次进入VPS的时候，会要求输入touchu ID，来保证服务器操作安全。
